# Data Mining Foundations on IMDb Dataset

## Overview
This project was developed for the *Data Mining: Foundations* course as part of my Master's degree in Data Science and Business Informatics at the University of Pisa.

The project focuses on the analysis of IMDb data through fundamental data mining techniques.

It covers key topics such as data understanding and preprocessing, outlier detection, clustering, classification, regression, and pattern mining, with the goal of exploring data structure and building predictive models.
